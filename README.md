## How to use

### Browser

```js
<script src="lib/WMFoo.js"></script>
<script>
console.log( WMFoo() );
</script>
```

### WebWorkers

```js
importScripts("lib/WMFoo.js");

console.log( WMFoo() );
```

### Node.js

```js
var WMFoo = require("lib/WMFoo.js");

console.log( WMFoo() );
```
